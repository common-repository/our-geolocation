== Description ==

Show your store or business or office location at Google map and let your visitor find you easily.

> * [Live Demo](http://www.wpcue.com/wordpress-plugins/geolocation/geolocation-demo/)

= Key Feature =

* Easy to use
* Address autocomplete suggestion for visitor
* Show details direction to your address

== Installation ==

Using the WordPress dashboard

1. Login to admin panel
2. Go to Plugins
3. Select Add New
4. Search Our Geolocation
5. Select Install Now
6. Select Activate Plugin
7. Click "Direction to Us" tab enter your location

Manual

1. Download the plugin and extract the files
2. Upload the directory "our-geolocation" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Geolocation" Tab created by this plugins for manage options

Use shortcode [our-geolocation]

Use WordPress function "do_shortcode" for use in template or theme PHP file.
`<?php echo do_shortcode( '[our-geolocation]' ); ?>`
